#  Python Labs

##  Лабораторная работа №1: Класс и инкапсуляция

###  Тема
Создание классов, инкапсуляция, свойства и магические методы

###  Задачи
-  Разработка классов транспортной системы (Driver, Route, Bus, Ticket)
-  Реализация закрытых атрибутов и инкапсуляции
-  Применение свойств @property (геттеры и сеттеры)
-  Переопределение магических методов __str__, __repr__, __eq__
-  Использование атрибутов класса и экземпляра
-  Добавление состояний объектов через Enum
-  Реализация поведения, зависящего от состояния
-  Вынесение валидации в отдельные методы

###  Цель работы
-  Освоить объявление пользовательских классов
-  Разобраться с инкапсуляцией (атрибуты экземпляра, закрытые поля)
-  Реализовать свойства (@property)
-  Переопределить магические методы (__str__, __repr__, __eq__)
-  Осознать разницу между атрибутами класса и экземпляра

###  Используемые техники
`class`, инкапсуляция, `@property`, магические методы, атрибуты класса, Enum, валидация данных, обработка исключений.

## Лабораторная работа №2: Класс-коллекция

### Тема
Создание класса-коллекции для хранения объектов, реализация методов добавления, удаления, поиска, сортировки и фильтрации

### Задачи
- Разработка класса-коллекции BusFleet для хранения автобусов
- Реализация методов add, remove, remove_at, get_all, clear
- Реализация методов поиска по атрибутам (find_by_id, find_by_route, find_by_driver, find_by_status)
- Применение магических методов __len__, __iter__, __getitem__, __contains__
- Реализация сортировки коллекции по различным критериям
- Реализация фильтрации с возвратом новой коллекции
- Защита от дубликатов при добавлении объектов
- Поддержка индексации и срезов (включая отрицательные индексы)

### Цель работы
- Реализовать класс-коллекцию для хранения объектов
- Освоить методы добавления, удаления и поиска элементов
- Реализовать магические методы __len__, __iter__, __getitem__
- Научиться сортировать и фильтровать коллекции
- Реализовать защиту от дубликатов

### Используемые техники
`class`, коллекции, магические методы, итераторы, сортировка, фильтрация, индексация, обработка исключений, повторное использование кода.

## Лабораторная работа №3: Наследование и иерархия классов

### Тема
Построение иерархии классов на основе базового класса Bus, реализация наследования и полиморфизма

### Задачи
- Создание базового класса Bus (перенос из ЛР-1)
- Разработка трех дочерних классов: TouristBus, CityBus, SchoolBus
- Добавление новых атрибутов в каждый дочерний класс
- Реализация новых методов в дочерних классах
- Переопределение методов базового класса (calculate_fare, get_bus_type, __str__)
- Использование super() для вызова конструктора базового класса
- Реализация полиморфного поведения (один метод - разная реализация)
- Интеграция с коллекцией BusFleet из ЛР-2
- Фильтрация коллекции по типу объектов

### Цель работы
- Освоить механизм наследования классов
- Научиться строить иерархию объектов
- Понять разницу между базовым и производным классами
- Научиться переиспользовать код
- Освоить переопределение методов
- Реализовать полиморфизм

### Используемые техники
`class`, наследование, `super()`, переопределение методов, полиморфизм, `isinstance()`, фильтрация по типу, магические методы, коллекции.
