# Ответы на вопросы 
## Вариант №5: Транспортная система (класс Bus)

---

## Вопрос 1. Что является сущностью?

**Какой объект реально существует в этой предметной области?**

В транспортной системе основной сущностью является:

| Сущность | Описание |
|----------|----------|
| **Bus (Автобус)** | Транспортное средство для перевозки пассажиров |

Автобус имеет свои уникальные характеристики (идентификатор, вместимость, маршрут, водитель, скорость, состояние) и поведение (выезд на маршрут, парковка, ускорение, торможение, обслуживание).

---

## Вопрос 2. Какие у него атрибуты?

**Каждый атрибут должен отвечать на вопрос: Это характеристика объекта или действие?**

### Класс Bus (Автобус)

| Атрибут | Тип | Характеристика/Действие |
|---------|-----|-------------------------|
| `_id` | Характеристика | Идентификатор автобуса (госномер) |
| `_capacity` | Характеристика | Вместимость (количество пассажиров) |
| `_route_number` | Характеристика | Номер маршрута |
| `_driver_name` | Характеристика | Имя водителя |
| `_speed` | Характеристика | Текущая скорость (км/ч) |
| `_status` | Характеристика | Состояние автобуса |
| `_last_maintenance` | Характеристика | Дата последнего технического обслуживания |
| `start_route()` | Действие | Выехать на маршрут |
| `park()` | Действие | Встать на парковку |
| `send_to_maintenance()` | Действие | Отправить на техническое обслуживание |
| `accelerate()` | Действие | Увеличить скорость |
| `brake()` | Действие | Уменьшить скорость |

## Вопрос 3. Какие инварианты?

**Инвариант — это правило, которое всегда должно быть верным.**

### Инварианты для Bus (Автобус):

| Инвариант | Пояснение |
|-----------|-----------|
| ✅ ID автобуса **не может быть пустым** | Длина от 1 до 10 символов |
| ✅ Вместимость **> 0** и **≤ 150** | Автобус не может вместить больше 150 пассажиров |
| ✅ Скорость **≥ 0** и **≤ MAX_SPEED** | MAX_SPEED = 100 км/ч |
| ✅ Если автобус **на парковке**, скорость **должна быть 0** | Нельзя парковаться на скорости |
| ✅ Если автобус **на ТО**, скорость **должна быть 0** | На обслуживании автобус стоит |
| ✅ Дата последнего ТО **всегда существует** | Устанавливается при создании и обновляется при ТО |
| ✅ Номер маршрута (если задан) **должен быть от 1 до 999** | Валидный диапазон номеров |
| ✅ Имя водителя (если задано) **должно быть не менее 2 символов** | Минимальная длина имени |

## Вопрос 4. Что значит “равенство”?

**Когда два объекта считаются равными?**

| Класс | Критерий равенства | Обоснование |
|-------|-------------------|-------------|
| **Bus** | `_id` (бортовой номер/госномер) | Каждый автобус имеет уникальный идентификатор |

**Примеры:**
- Два автобуса равны, если у них **совпадают ID** (`bus_id`)
- Остальные атрибуты (маршрут, водитель, скорость, состояние) могут отличаться

```python
# Пример из demo.py
bus1 == bus2  # False (разные ID: А123ВС и В456СМ)
bus1 == bus1  # True (один и тот же объект)
```

## Вопрос 5. Есть ли состояние?

**Есть ли у объекта состояние?**

### Состояния для Bus (Автобус):

| Состояние | Описание | Разрешенные действия | Запрещенные действия |
|-----------|----------|---------------------|---------------------|
| **ON_ROUTE** | На маршруте | ускорение, торможение, изменение скорости | смена маршрута, парковка (если скорость > 0) |
| **ON_PARKING** | На парковке | выезд на маршрут, отправка на ТО | ускорение, изменение скорости (если скорость > 0) |
| **MAINTENANCE** | На ТО | завершение ТО | ускорение, изменение скорости, выезд на маршрут |
