**Атрибуты (закрытые поля)**

### 🚫 Поведение, зависящее от состояния

**Для автобуса:**
| Действие | В каком состоянии запрещено |
|----------|-----------------------------|
| Ускорение | На парковке, на ТО |
| Изменение скорости | На ТО |
| Смена маршрута | В пути |
| Выезд на маршрут | Без назначенного маршрута |

**Для билета:**
| Действие | В каком состоянии запрещено |
|----------|-----------------------------|
| Покупка | Уже продан |
| Использование | Не продан, уже использован |

### ✅ Валидация данных

**Для водителя:**
| Поле | Проверка |
|------|----------|
| Имя | минимум 2 символа |
| Номер прав | минимум 3 символа |
| Стаж | от 0 до 60 лет |
| Категория | A, B, C или D |

**Для маршрута:**
| Поле | Проверка |
|------|----------|
| Номер | от 1 до 999 |
| Начальный пункт | минимум 2 символа |
| Конечный пункт | минимум 2 символа |
| Расстояние | от 1 до 1000 км |
| Длительность | от 1 до 1440 мин |

**Для автобуса:**
| Поле | Проверка |
|------|----------|
| ID | от 1 до 10 символов |
| Вместимость | от 1 до 150 |
| Скорость | от 0 до 100 км/ч |
| Статус | из допустимых |

**Для билета:**
| Поле | Проверка |
|------|----------|
| ID | не пустой |
| Цена | от 1 до 50000 руб |
| Статус | из допустимых |

### 🔒 Инкапсуляция

Все атрибуты классов являются **закрытыми** (начинаются с `_`). Доступ к ним осуществляется через **свойства** (`@property`), что позволяет контролировать чтение и запись данных с валидацией.

### 📝 Магические методы

| Метод | Назначение |
|-------|------------|
| `__str__` | Для пользователя (красивое описание с эмодзи) |
| `__repr__` | Для разработчика (показывает конструктор) |
| `__eq__` | Сравнение объектов по уникальным полям |
