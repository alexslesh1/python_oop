### Класс Bus (Автобус)

#### Атрибуты (закрытые поля)
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `_id` | str | Идентификатор автобуса |
| `_capacity` | int | Вместимость (1-150) |
| `_route_number` | int/str | Номер маршрута |
| `_driver_name` | str | Имя водителя |
| `_speed` | float | Текущая скорость (0-100) |
| `_status` | BusStatus | Состояние автобуса |
| `_last_maintenance` | datetime | Дата последнего ТО |

#### Атрибуты класса
| Атрибут | Значение | Описание |
|---------|----------|----------|
| `_total_buses` | счетчик | Количество созданных автобусов |
| `_MAX_SPEED` | 100 | Максимальная скорость |

#### Свойства (@property)
| Свойство | Геттер | Сеттер | Описание |
|----------|--------|--------|----------|
| `bus_id` | ✅ | ❌ | Только чтение |
| `capacity` | ✅ | ❌ | Только чтение |
| `route_number` | ✅ | ✅ | С проверкой (нельзя сменить в пути) |
| `driver_name` | ✅ | ✅ | Чтение/запись |
| `speed` | ✅ | ✅ | С проверкой состояния |
| `status` | ✅ | ❌ | Только чтение |
| `is_broken` | ✅ | ❌ | Логическое состояние (нужда в ТО) |

#### Магические методы
| Метод | Описание |
|-------|----------|
| `__str__` | Красивое описание для пользователя |
| `__repr__` | Официальное представление для разработчика |
| `__eq__` | Сравнение объектов по ID |

#### Бизнес-методы
| Метод | Описание |
|-------|----------|
| `start_route()` | Выехать на маршрут |
| `park()` | Поставить на парковку |
| `send_to_maintenance()` | Отправить на ТО |
| `accelerate(inc)` | Увеличить скорость |
| `brake(dec)` | Уменьшить скорость |
| `is_broken` | Проверить необходимость ТО (свойство) |

### Состояния объекта

| Перечисление | Значения | Описание |
|--------------|----------|----------|
| **BusStatus** | `ON_ROUTE` | На маршруте |
| | `ON_PARKING` | На парковке |
| | `MAINTENANCE` | На обслуживании |

### Поведение, зависящее от состояния

| Действие | В каком состоянии запрещено |
|----------|-----------------------------|
| Ускорение (`accelerate`) | На парковке, на ТО |
| Изменение скорости (`speed.setter`) | На ТО |
| Смена маршрута (`route_number.setter`) | В пути (`ON_ROUTE`) |
| Выезд на маршрут (`start_route`) | Без назначенного маршрута |
| Парковка (`park`) | При скорости > 0 |

### Валидация данных (файл `validators.py`)

| Поле | Проверка | Функция |
|------|----------|---------|
| ID автобуса | от 1 до 10 символов | `validate_bus_id()` |
| Вместимость | от 1 до 150 | `validate_capacity()` |
| Скорость | от 0 до 100 км/ч | `validate_speed()` |
| Статус | должен быть типа BusStatus | `validate_status()` |
| Номер маршрута | от 1 до 999 (если указан) | `validate_route_number()` |
| Имя водителя | минимум 2 символа (если указано) | `validate_driver_name()` |

### Атрибуты класса

| Атрибут | Назначение |
|---------|------------|
| `Bus._total_buses` | Количество созданных автобусов |
| `Bus._MAX_SPEED` | Максимальная скорость для всех автобусов (100 км/ч) |

### Инкапсуляция

Все атрибуты класса являются **закрытыми** (начинаются с `_`). Доступ к ним осуществляется через **свойства** (`@property`), что позволяет контролировать чтение и запись данных с валидацией.

### Магические методы

| Метод | Назначение |
|-------|------------|
| `__str__` | Для пользователя (красивое описание) |
| `__repr__` | Для разработчика (показывает конструктор объекта) |
| `__eq__` | Сравнение объектов по уникальному полю `bus_id` |
