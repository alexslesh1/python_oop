### Класс BusFleet (Коллекция автобусов)

#### Атрибуты (закрытые поля)
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `_items` | list | Внутренний список для хранения автобусов |

#### Базовые методы (оценка 3)
| Метод | Описание |
|-------|----------|
| `add(bus)` | Добавить автобус в коллекцию (с проверкой типа и дубликатов) |
| `remove(bus)` | Удалить автобус из коллекции |
| `remove_at(index)` | Удалить автобус по индексу |
| `get_all()` | Вернуть список всех автобусов |
| `clear()` | Очистить коллекцию |

#### Методы поиска (оценка 4)
| Метод | Описание |
|-------|----------|
| `find_by_id(bus_id)` | Найти автобус по ID |
| `find_by_route(route_number)` | Найти все автобусы по номеру маршрута |
| `find_by_driver(driver_name)` | Найти все автобусы по имени водителя |
| `find_by_status(status)` | Найти все автобусы по состоянию |

#### Методы фильтрации (оценка 5)
| Метод | Описание |
|-------|----------|
| `get_on_route()` | Вернуть коллекцию автобусов на маршруте |
| `get_needs_maintenance()` | Вернуть коллекцию автобусов, требующих ТО |
| `get_by_min_speed(min_speed)` | Вернуть коллекцию автобусов со скоростью >= min_speed |

#### Методы сортировки (оценка 5)
| Метод | Описание |
|-------|----------|
| `sort_by_id()` | Сортировка по ID |
| `sort_by_route()` | Сортировка по номеру маршрута |
| `sort_by_speed()` | Сортировка по скорости |
| `sort(key_func)` | Универсальная сортировка |

#### Магические методы
| Метод | Описание |
|-------|----------|
| `__len__()` | Возвращает количество автобусов в коллекции |
| `__getitem__(index)` | Доступ по индексу (поддерживает срезы и отрицательные индексы) |
| `__iter__()` | Итератор для использования в цикле for |
| `__str__()` | Строковое представление коллекции |
| `__repr__()` | Официальное представление для разработчика |
| `__contains__(bus)` | Проверка наличия автобуса в коллекции |

### Защита от дубликатов

При добавлении автобуса проверяется уникальность `bus_id`. Если автобус с таким ID уже существует в коллекции, выбрасывается исключение `ValueError`. Это предотвращает появление дубликатов и обеспечивает целостность данных.

### Индексация и срезы

Класс `BusFleet` поддерживает стандартные операции доступа к элементам:

| Операция | Результат |
|----------|-----------|
| `fleet[index]` | Получение элемента по индексу (положительному или отрицательному) |
| `fleet[start:end]` | Срез коллекции (возвращает новую коллекцию) |
| `fleet[-1]` | Получение последнего элемента |
| `fleet[-2]` | Получение предпоследнего элемента |

### Сортировка

Коллекция поддерживает сортировку по различным критериям:

| Метод | Критерий сортировки |
|-------|---------------------|
| `sort_by_id()` | Алфавитный порядок по ID автобуса |
| `sort_by_speed()` | Возрастание по текущей скорости |
| `sort_by_route()` | Числовой порядок по номеру маршрута (None в конец) |

### Фильтрация

Методы фильтрации возвращают **новую коллекцию** (не изменяют исходную):

| Метод | Условие отбора |
|-------|----------------|
| `get_on_route()` | Автобусы со статусом "На маршруте" |
| `get_needs_maintenance()` | Автобусы, требующие технического обслуживания (is_broken = True) |
| `get_by_min_speed(min_speed)` | Автобусы со скоростью не ниже указанной |

### Демонстрационные сценарии

В файле `demo.py` реализованы четыре сценария:

| Сценарий | Что демонстрируется |

### Магические методы в действии

| Метод | Пример использования | Результат |
|-------|---------------------|-----------|
| `__len__` | `len(fleet)` | Количество автобусов в коллекции |
| `__getitem__` | `fleet[0]`, `fleet[-1]`, `fleet[1:4]` | Доступ по индексу или срез |
| `__iter__` | `for bus in fleet:` | Перебор всех автобусов |
| `__contains__` | `bus in fleet` | Проверка наличия автобуса |
| `__str__` | `print(fleet)` | Красивый вывод коллекции |
| `__repr__` | `repr(fleet)` | Отладочное представление |
