## 2. Описание стратегий и функций

### Стратегии сортировки

| Функция | Назначение |
|---------|------------|
| `by_bus_id(bus)` | Сортировка по ID автобуса (алфавитный порядок) |
| `by_capacity(bus)` | Сортировка по вместимости (от меньшей к большей) |
| `by_speed(bus)` | Сортировка по текущей скорости |
| `by_route_number(bus)` | Сортировка по номеру маршрута (None в конец) |
| `by_comfort_level(bus)` | Сортировка по уровню комфорта (для туристических) |
| `by_passenger_load(bus)` | Сортировка по загруженности (для городских) |

### Стратегии фильтрации

| Функция | Назначение |
|---------|------------|
| `is_on_route(bus)` | Фильтр: автобус на маршруте |
| `is_on_parking(bus)` | Фильтр: автобус на парковке |
| `needs_maintenance_filter(bus)` | Фильтр: автобус требует ТО |
| `is_tourist_bus(bus)` | Фильтр: только туристические автобусы |
| `is_city_bus(bus)` | Фильтр: только городские автобусы |
| `is_school_bus(bus)` | Фильтр: только школьные автобусы |

### Фабрики функций

| Фабрика | Назначение |
|---------|------------|
| `speed_above_threshold(threshold)` | Создаёт фильтр для автобусов со скоростью выше порога |
| `capacity_below_threshold(threshold)` | Создаёт фильтр для автобусов с вместимостью ниже порога |

### Callable-объекты (паттерн Стратегия)

| Класс | Назначение |
|-------|------------|
| `SortByIdStrategy` | Callable-объект для сортировки по ID |
| `SortBySpeedStrategy` | Callable-объект для сортировки по скорости |
| `SortByRouteStrategy` | Callable-объект для сортировки по маршруту |
| `NeedsMaintenanceStrategy` | Callable-объект для фильтрации по необходимости ТО |
| `DiscountStrategy` | Callable-объект для применения скидки к стоимости проезда |
| `ActivateStrategy` | Callable-объект для активации/деактивации автобусов |

### Методы расширенной коллекции

| Метод | Назначение |
|-------|------------|
| `sort_by(key_func, reverse)` | Сортировка коллекции по переданной функции-ключу |
| `filter_by(predicate)` | Фильтрация коллекции по предикату (возвращает новую коллекцию) |
| `apply(func)` | Применение функции ко всем элементам коллекции |
| `map(func)` | Преобразование элементов с использованием встроенной `map()` |

## 3. Реализация

### Базовые функции-стратегии

```python
def by_bus_id(bus):
    return bus.bus_id

def by_capacity(bus):
    return bus.capacity

def by_route_number(bus):
    return bus.route_number if bus.route_number is not None else float('inf')
```

### Фабрика функций (замыкание)
``` python
def speed_above_threshold(threshold):
    def filter_fn(bus):
        return bus.speed >= threshold
    return filter_fn
```

### Callable-объект (паттерн Стратегия)
``` python
class DiscountStrategy:
    def __init__(self, discount=0.1):
        self.discount = discount
    
    def __call__(self, bus):
        fare = bus.calculate_fare()
        return fare * (1 - self.discount)
```

### Расширенная коллекция
``` python
class ExtendedBusFleet(BusFleet):
    def sort_by(self, key_func, reverse=False):
        self._items.sort(key=key_func, reverse=reverse)
        return self
    
    def filter_by(self, predicate):
        filtered = ExtendedBusFleet()
        filtered._items = [item for item in self._items if predicate(item)]
        return filtered
    
    def apply(self, func):
        return [func(item) for item in self._items]
```
